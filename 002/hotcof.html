:: 카푸치노 (cappuccono) ::

재료
에스프레소 커피추출액1컵,우유1/3컵,계피가루 1/4작은술,계피스틱
만들기
1. 에스프레소 커피가루에 계피가루를 섞어 커피를 추출한다.
2. 손잡이가 달린 냄비에 우유를 넣고 끊기 직전까지 데운다. 충분히 거품을 내기 위해 블렌더로 몇초간 저어준다.
3. 계피스틱을 꽂고 커피를 부은후 그위에 거품 낸 우유를 조심따른다.
4. 휘핑크림을 얹고 계피가루를 뿌린다.
:: 에스프레소 커피() ::

재료
에스프레소 커피 추출액 1컵, 설탕, 레몬껍질
만들기
1.일인용 에스프레소 커피 기구에 커피가루를 다져 넣고 끊인다.
2. 에스프레소 잔에 커피를 따른다. 컵에 레몬껍질을 미리 넣어두기도 한다.
3.취향에 따라 설탕을 넣어 마신다
:: 카페오레 () ::

재료
커피 추출액 1/2컵, 우유1/2컵
만들기
1. 손잡이가 달리 냄비에 우유를 끓기 직전까지 데운다. 데우면서 스푼으로 저으면 거품이 생긴다.
2. 뜨거운 커피와 우유를 일대일의 비율로 컵에 붓는다.
3. 즉시 마신다. 취향에 따라 설탕을 넣어도 좋다
:: 비엔나 커피() ::

재료
커피(원두커피를 걸러 만든 것), 생크림, 설탕, 레몬즙, 화이트 초콜릿
만들기
1. 볼에 생크림,설탕,레몬즙을 넣고 거품기로 한참동안저어 거품을 낸다.
2. 화이트 초콜릿을 껍질깎기로 깎아서 얇게 만든다.
3. 따뜻하게 데운 컵에 커피를 7부 정도만 붓고 생크림을 얹는다. 그 위에 얇게 썬 초콜릿을 뿌린다
