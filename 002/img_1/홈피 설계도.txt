홈페이지 설계도

주제: 커피

index.html : 첫 홈페이지 이미지 화면 

  ->main.html : 기본 메인화면 (공지사항, 홈페이지소개등..)
                (프레임 3구분 top.html, left.html, right.html)

      ( left 프레임)
      ->profile.html : 자기소개란
          ->myphoto.html : 나의 사진들

      ->knowcof.html : "커피알기" 메뉴 (프레임 상하2구분)
          ->hiscof.html : 커피의 역사 (연도별 변천...)
          ->crecof.html : 커피의 생산지 (콜롬비아,자메이카,인도네시아...)
          ->delcof.html : 커피의 맛 (볶는과정, 끓이는 방법에 따른맛...)
          ->woncof.html : 원두 커피 (좋은원두선별, 원두향기보존...)
          ->detcof.html : 커피 성분 (커피의 영양...)

      ->bobbcof.html : "커피 끊이기" 메뉴 (커피끊이기 기본 상식)
          ->hotcof.html : 따뜻한 커피 (따뜻한 커피 종류별 끊이기 방법)
          ->colcof.html : 시원한 커피 (시원한 커피 종류별 끊이기 방법)

      ->instcof.html : "커피 기구" 메뉴 (커피 가는 기구, 여과기... 소개)

      ->withcof.html : "커피와 함께" 메뉴 (커피와 함께 하면서...)
          ->cookcake.html : 쿠키와 케잌 (커피와 함께 먹으면 좋은 음식)
          ->beauty.html : 아름다운 시 (분위기 있는 아름다운 시)

      ->(게시판1 링크) : "about 커피" 메뉴 (커피와 얽힌 사연, 관한글)
      
      ->(게시판2 링크) : "방명록" 메뉴

      ( top 프레임)
      ->(right.html 링크) : "go home" 메뉴 (처음메인화면 돌아가기버튼)
